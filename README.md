# Activity 11 - Menus Project (Laravel)

## Description
This project was developed using Laravel and Blade template engine.

The objective of this activity is to understand and implement Blade Template Inheritance by creating a main layout and multiple views that extend from it.

## Features
- Main layout view (Blade)
- Navigation menu with three options:
  - Home
  - Photos
  - Contact
- Active menu highlighting depending on the current route
- Three independent views with different content
- Shared footer displayed across all views
- Use of Blade directives (@extends, @section, @yield)

## Technologies Used
- Laravel
- Blade Template Engine
- PHP
- HTML / CSS

## Project Structure
- layouts/app.blade.php → Main layout
- home.blade.php → Home section
- photos.blade.php → Photos section
- contact.blade.php → Contact section
